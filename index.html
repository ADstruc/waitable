The Wait Is Over

The Waitable button is a jQuery plugin that allows you to quickly create a unified waiting state look for your buttons that trigger AJAX actions. Providing a function which returns a jQuery promise, waitable button takes care of the boilerplate of setting up waiting indicators, double submit prevention, and more!*

* Actually, that's all it does. But quick, watch this!
